browser_input関数
Browserの指定項目に入力を行う

構文
int browser_input(string BrowserKeyName, string InputData, string Element, int SearchType)

パラメタ
BrowserKeyName
タイプ：string
browser_navigate関数で取得した戻り値

InputData
タイプ：string
指定項目に入力を行いたいテキストデータ

Element
タイプ：string
入力を行いたい項目名

SearchType
タイプ：string
項目検索タイプ

戻り値
タイプ：int
関数が成功した場合、戻り値は0になります。エラーの場合には0以外になります。
