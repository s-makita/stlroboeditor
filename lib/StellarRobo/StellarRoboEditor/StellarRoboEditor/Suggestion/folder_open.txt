folder_open関数
指定FolderをExplorerで開く

構文
int folder_open(string PathName, int WindowStyle, int ExitsWait, int ExitsWaitTime)

パラメタ
PathName
タイプ：string
開きたいFolderへのパス

WindowStyle
タイプ：int
起動時のWindowスタイルを指定

ExitsWait
タイプ：int
アプリが終了するまで、待つかを指定

ExitsWaitTime
タイプ：int
アプリが終了する待つ時間を指定

戻り値
タイプ：int
関数が成功した場合、戻り値は0になります。エラーの場合には0以外になります。
